## 2.6 符号运算

`sym (syms)` 定义符号变量（表达式）

`diff(f,x,n)` 表示 f 关于 x 求 n 阶导数

`int(f, r, x0, x1)` 表示 f 关于 r 求从 x0 到 x1 的定积分

## 2.7 复数运算基础

`subs` 对符号函数中自变量进行赋值
e.g. subs(f, {x,y}, {-1,2}) 对函数 f 中自变量 x 和 y 分别赋值 -1 和 2

`polar()` 绘制极坐标图
